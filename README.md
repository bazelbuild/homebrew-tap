# Bazel Homebrew Tap
This repository contains a collection of Homebrew (aka, Brew) "formulae" for Bazel

## Installing Bazel

```
$ brew install bazel
```

## Upgrading Bazel

```
$ brew upgrade bazel
```

## Installing buildozer

```sh
$ brew install buildozer
```

## Upgrading buildozer

```sh
$ brew upgrade buildozer
```
